## 📖 Description

This project predicts whether a student will pass or fail using Machine Learning based on study hours, attendance, and previous marks.

---

## 🚀 Features

* Predict Pass/Fail
* Confidence Score
* Performance Suggestions
* Simple User Interface

---

## 🛠️ Installation

pip install pandas scikit-learn streamlit

---

## ▶️ Usage

python model.py
python -m streamlit run app.py
