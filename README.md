## 📖 Description

This project predicts whether a student will pass or fail using Machine Learning based on study hours, attendance, and previous marks.

---

## 🚀 Features

* Predict Pass/Fail
* 
* Confidence Score
* 
* Performance Suggestions
* 
* Simple User Interface

---

## 🛠️ Installation

pip install pandas scikit-learn streamlit

---

## ▶️ Usage

python model.py

python -m streamlit run app.py

---

▶️ How to Run (Step-by-Step)

Step 1: Clone or Download the Project

Download the project files or clone from GitHub.

Step 2: Install Required Libraries

pip install pandas scikit-learn streamlit

Step 3: Train the Model

python model.py

👉 This will create: student_model.pkl

Step 4: Run the Application

python -m streamlit run app.py

Step 5: Open in Browser

After running, you will see a link like:

http://localhost:8501

👉 Open it in your browser to use the app.
